Java Tutorial

What is Java?

Java is a programming language and a platform. Java is a high-level, robust, object-oriented, and secure programming language.
Java was developed by Sun Microsystems in 1995, and James Gosling is known as its father. Initially called Oak, the name was later changed to Java

Java Data Types

Java data types are categorized into:

  • Integer Types: Stores whole numbers (e.g., byte, short, int, long).
  • Floating Point Types: Represents numbers with decimals (e.g., float, double).

Syntax:


            int myNum = 5;               // Integer (whole number)
            float myFloatNum = 5.99f;    // Floating point number
            boolean myBool = true;       // Boolean
            char myLetter = 'D';         // Character
            String myText = "Hello";     // String
